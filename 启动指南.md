# 网站启动指南

## 启动方式

### 方式一：使用批处理文件（推荐）
双击运行项目根目录下的 `启动项目.bat.txt` 文件（如果已重命名为 `.bat`）

### 方式二：手动启动

1. **打开终端/命令行**
   - 在项目目录 `d:\游戏网站1\wegogame.net-main` 中打开终端
   - 或使用 VS Code/Cursor 的集成终端

2. **安装依赖（首次运行需要）**
   ```bash
   npm install
   ```

3. **启动开发服务器**
   ```bash
   npm run dev
   ```

4. **访问网站**
   - 服务器启动后，会显示类似以下信息：
     ```
     VITE v5.x.x  ready in xxx ms
     
     ➜  Local:   http://localhost:5173/
     ➜  Network: use --host to expose
     ```
   - 在浏览器中打开 `http://localhost:5173`

## 注意事项

- 如果遇到权限问题，请以**非管理员身份**运行 Cursor/终端
- 确保已安装 Node.js（建议 v16+）
- 如果端口 5173 被占用，Vite 会自动使用下一个可用端口

## 修改代码后

- 开发服务器支持热重载，修改代码后会自动刷新浏览器
- 无需手动重启服务器
