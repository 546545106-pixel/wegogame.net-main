# 60 款 H5 小游戏部署说明

## 功能

1. **删除网站内现有全部游戏**，只保留「Html5小游戏JavaScript源码」里这 60 款。
2. **解压并部署**：从 `Html5小游戏JavaScript源码/60套H5小游戏源码大合集/` 的 60 个 `.rar` 解压到 `public/games/game_01`～`game_60`。
3. **广告接入**：移除游戏内第三方广告，注入本站 `/game-ad-api.js`，统一走你在 `src/utils/adManager.js` 里配置的 AdSense（ca-pub-5319587106206709）。
4. **缩略图**：使用 `1-60套预览图` 中的 01.jpg～60.png 作为各游戏封面。
5. **重写列表**：`src/data/games.js` 仅包含这 60 款游戏，网站首页只展示它们。

## 使用方式

### 方式一：批处理（推荐）

双击运行 **`部署60游戏.bat`**。

### 方式二：命令行

在项目根目录执行：

```bash
cd "d:\游戏网站2\wegogame.net-main"
python 部署60游戏.py
```

## 环境要求

- **Python 3.6+**
- **7-Zip**：用于解压 `.rar`。若未安装，请安装后重试，或先手动把 60 个 RAR 解压到 `_60games_extracted/01`～`_60games_extracted/60`，再运行脚本。

## 广告配置

- 站内已使用 **ca-pub-5319587106206709**（见 `index.html`、`AdSense接入指南.md`）。
- 游戏内通过 `/game-ad-api.js` 请求 `game_01`～`game_60` 的广告位。
- 在 **`src/utils/adManager.js`** 中，把常量 **`GAME_INTERSTITIAL_SLOT`** 的值从 `'your-game-interstitial-slot-id'` 改成你在 AdSense 后台创建的游戏插屏广告位 ID，即可在这 60 款游戏里展示你的广告。

## 部署后

1. 运行 `npm run dev`，在浏览器打开 `http://localhost:5173` 查看首页与游戏列表。
2. 逐一点进几款游戏，确认能正常进入、操作、加载。
3. 若某款游戏黑屏或报错，多为该游戏源码路径/入口与脚本假设不同，可针对该游戏目录再排查。

## 备份与回滚

- 运行前会自动把当前 **`src/data/games.js`** 备份为 **`src/data/games.js.backup_60`**。
- 若要恢复原来的游戏列表，把 `games.js.backup_60` 重命名回 `games.js` 即可（恢复后需自行恢复 `public/games/` 下的旧游戏文件）。
