# 合并所有游戏说明

## 功能说明

此脚本用于将之前的106个外部游戏（来自gamemonetize.com）和新的423个本地游戏（部署在`/games/`目录）合并到一个`games.js`文件中。

## 使用方法

### 方法1：双击批处理文件（推荐）

1. 双击运行 `合并所有游戏.bat`
2. 脚本会自动执行合并操作
3. 完成后查看输出信息

### 方法2：使用Python直接运行

```bash
python 合并所有游戏.py
```

## 脚本功能

1. **读取外部游戏**：从 `src/data/games.js.backup` 或 `xixi-games/src/data/games.js` 读取106个外部游戏
2. **读取本地游戏**：从 `src/data/games.js` 读取423个本地游戏
3. **合并去重**：根据游戏ID合并，避免重复
4. **生成新文件**：生成包含所有游戏的 `games.js` 文件
5. **自动备份**：合并前会自动备份当前文件为 `games.js.backup2`

## 输出结果

合并完成后，`src/data/games.js` 将包含：
- 106个外部游戏（使用 `https://html5.gamemonetize.com/...` URL）
- 423个本地游戏（使用 `/games/xxx/index.html` URL）
- **总计：529个游戏**

## 注意事项

1. **备份**：脚本会自动备份当前文件，但建议手动备份重要数据
2. **Python要求**：需要安装Python 3.x
3. **文件路径**：确保所有文件路径正确
4. **ID冲突**：如果新旧游戏有相同ID，新游戏会覆盖旧游戏

## 验证合并结果

合并完成后，可以：
1. 检查 `src/data/games.js` 文件大小
2. 在网站首页查看游戏数量
3. 测试几个外部游戏和本地游戏是否都能正常加载

## 故障排除

### 问题1：找不到Python
**解决**：安装Python 3.x，下载地址：https://www.python.org/downloads/

### 问题2：找不到备份文件
**解决**：检查 `src/data/games.js.backup` 或 `xixi-games/src/data/games.js` 是否存在

### 问题3：合并后游戏数量不对
**解决**：
- 检查是否有ID冲突
- 查看脚本输出的详细信息
- 手动检查 `games.js` 文件内容

## 技术支持

如有问题，请检查：
1. Python版本是否正确
2. 文件路径是否正确
3. 文件编码是否为UTF-8
