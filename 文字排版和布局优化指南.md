# 文字排版和布局优化实施指南

## 优化概述

本次优化主要针对文字排版层级和页面布局进行了全面优化，包括：
- 文字层级规范化（标题、分类标题、游戏名称、辅助文字、按钮）
- 网格布局优化（PC端4列，移动端2列）
- 分类标签区布局
- 页面容器宽度限制和居中对齐
- 响应式适配

## 布局规范

### 1. 页面容器
```css
.ui-optimized-container {
  max-width: 1200px;
  margin: 0 auto;
  padding: 0 20px; /* PC端 */
}
@media (max-width: 768px) {
  .ui-optimized-container {
    padding: 0 16px; /* 移动端 */
  }
}
```
**效果：** 页面内容最大宽度1200px，居中对齐，左右自适应留白

---

### 2. 游戏列表网格布局

**PC端（>768px）：**
- 每行4个卡片
- 间距20px
- 使用 `:md="6" :lg="6" :xl="6"`（Element Plus的24栅格系统，6列 = 4个）

**移动端（≤768px）：**
- 每行2个卡片
- 间距16px
- 使用 `:xs="12" :sm="12"`（12列 = 2个）

**平板端（769px-1024px）：**
- 每行3个卡片
- 通过CSS Grid自动调整

---

### 3. 分类标签区

```css
.ui-optimized-category-tabs {
  display: flex;
  flex-direction: row;
  gap: 12px;
  flex-wrap: wrap;
}

.ui-optimized-category-tab {
  padding: 8px 16px;
  border-radius: 6px;
  background-color: #F5F7FA;
  color: #333333;
}

.ui-optimized-category-tab.active {
  background-color: #4A90E2;
  color: #FFFFFF;
}
```

**使用示例：**
```vue
<div class="ui-optimized-category-tabs">
  <button 
    class="ui-optimized-category-tab"
    :class="{ active: activeTab === 'all' }"
    @click="activeTab = 'all'"
  >
    全部
  </button>
  <button 
    class="ui-optimized-category-tab"
    :class="{ active: activeTab === 'puzzle' }"
    @click="activeTab = 'puzzle'"
  >
    休闲益智
  </button>
</div>
```

## 注意事项

1. **Element Plus栅格系统：** 使用24栅格，6列 = 4个卡片（24/6=4）
2. **间距统一：** PC端使用20px，移动端使用16px
3. **文字层级：** 严格按照规范使用对应的样式类
4. **容器嵌套：** 确保 `ui-optimized-container` 正确包裹内容
5. **响应式测试：** 在不同设备上测试布局效果
