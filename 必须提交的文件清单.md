## ⚠️ 重要提示

1. **游戏文件是必需的**
   - `public/games/` 目录包含所有游戏文件
   - 这些文件必须提交，否则网站上的游戏无法运行
   - 如果文件太大，考虑使用 Git LFS（Large File Storage）

2. **不要提交 node_modules**
   - 这些文件会通过 `npm install` 自动安装
   - 提交它们会让仓库变得非常大且不必要

3. **不要提交 dist**
   - 这是构建输出，会在 Cloudflare Pages 上自动生成
   - 提交它没有意义且会增加仓库大小

4. **确保 .gitignore 正确**
   - 如果 `.gitignore` 配置正确，Git 会自动忽略不需要的文件
   - 使用 `git status` 检查，确保只显示需要提交的文件
