### 2. 按钮点击事件

**位置：** `src/views/DetailPage/index.vue`

**代码：**
```javascript
const toPlay = () => {
  isLoading.value = true;
  show.value = true;
};

const handleIframeLoad = () => {
  setTimeout(() => {
    isLoading.value = false;
  }, 500);
};
```

**HTML绑定：**
```vue
<button class="play ui-optimized-button" @click="toPlay">Play Now</button>
<iframe @load="handleIframeLoad" />
```

---

### 3. 分类标签切换事件

**位置：** `src/components/CategoryTabs.vue`

**代码：**
```javascript
const handleTabClick = (value) => {
  activeTab.value = value
  emit('change', value)
}
```

**HTML绑定：**
```vue
<button
  class="ui-optimized-category-tab"
  :class="{ active: activeTab === tab.value }"
  @click="handleTabClick(tab.value)"
>
```
