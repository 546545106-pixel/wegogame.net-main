# 431个H5游戏部署和翻译说明

## 功能说明

本脚本会自动完成以下任务：
1. ✅ 扫描 `431套H5小游戏源码大合集 带网页导航\games\` 目录下的所有游戏文件夹
2. ✅ 将游戏复制到 `public/games/` 目录
3. ✅ 自动检测并翻译所有中文内容为英文
4. ✅ 更新 `src/data/games.js` 文件，添加所有游戏数据

## 使用方法

### 方法一：使用批处理文件（推荐）

1. 双击运行 `部署431游戏.bat` 文件
2. 等待脚本执行完成（可能需要几分钟到十几分钟，取决于游戏数量）
3. 查看输出信息确认部署成功

### 方法二：使用Python命令

1. 打开命令提示符（CMD）或PowerShell
2. 切换到项目目录：
   ```bash
   cd "d:\游戏网站1\wegogame.net-main"
   ```
3. 运行脚本：
   ```bash
   python 部署431游戏并翻译.py
   ```

## 翻译功能说明

### 翻译范围

脚本会自动翻译以下文件中的中文内容：
- **HTML文件** (`.html`, `.htm`): 翻译title、meta描述、可见文本
- **JavaScript文件** (`.js`): 翻译字符串字面量中的中文（跳过URL、路径等）
- **文本文件** (`.txt`, `.xml`): 翻译所有可见文本

### 翻译策略

1. **字典翻译**：使用预定义的常见游戏术语翻译字典
2. **智能识别**：自动跳过代码、URL、文件路径等，只翻译用户可见的文本
3. **安全保护**：不会修改游戏逻辑代码，只翻译显示文本

### 已翻译的常见术语

- 开始游戏 → Start Game
- 游戏结束 → Game Over
- 得分/分数 → Score
- 等级/关卡 → Level
- 设置 → Settings
- 等等...

## 注意事项

1. **备份重要文件**：
   - 脚本会自动备份 `src/data/games.js` 为 `games.js.backup`
   - 建议在运行前手动备份整个项目

2. **处理时间**：
   - 431个游戏的处理可能需要10-30分钟
   - 请耐心等待，不要中断脚本

3. **翻译质量**：
   - 脚本使用字典翻译，可能无法覆盖所有中文
   - 部分游戏中的中文可能需要手动检查和调整

4. **游戏测试**：
   - 部署完成后，建议测试几个游戏确保正常运行
   - 检查游戏是否能正常加载和游玩

5. **文件大小**：
   - 431个游戏文件可能占用较大空间
   - 确保有足够的磁盘空间

## 故障排除

### 问题1：Python未安装
**解决方案**：安装Python 3.6或更高版本
- 下载地址：https://www.python.org/downloads/
- 安装时勾选"Add Python to PATH"

### 问题2：找不到游戏文件夹
**解决方案**：确认游戏文件夹路径正确
- 路径应为：`431套H5小游戏源码大合集 带网页导航\games\`

### 问题3：翻译不完整
**解决方案**：
- 脚本使用字典翻译，可能无法覆盖所有中文
- 可以手动编辑游戏文件补充翻译
- 或使用在线翻译工具辅助翻译

### 问题4：游戏无法运行
**解决方案**：
- 检查游戏文件是否完整复制
- 查看浏览器控制台错误信息
- 确认游戏路径正确
